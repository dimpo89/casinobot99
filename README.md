# Telegram Casino Bot 🎰

Полноценный Telegram бот для азартных игр с множеством функций!

## 🎮 Возможности

- **6 игр**: Слоты, Кости, Рулетка, Блэкджек, Plinko, Mines
- **Provably Fair**: Все игры с криптографической проверкой честности
- **RTP 98.2%** (отображаемый) / **76.82%** (реальный)
- **Джекпот**: 5% от каждой ставки идет в общий джекпот
- **Турниры**: Регулярные турниры с призовыми фондами
- **Реферальная система**: 5% от ставок рефералов
- **Ежедневные бонусы**: Система Streak-наград
- **Кран**: Бесплатные звезды каждые N минут
- **Админ панель**: Полное управление ботом

## 🚀 Быстрый старт

1. Создайте бота у [@BotFather](https://t.me/botfather)
2. Получите токен
3. Нажмите кнопку "Deploy to Render" ниже

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy)

## ⚙️ Ручной деплой

1. Форкните этот репозиторий
2. Зарегистрируйтесь на [render.com](https://render.com)
3. Создайте новый Web Service
4. Подключите репозиторий
5. Добавьте переменные окружения:
   - `BOT_TOKEN`: токен от @BotFather
   - `ADMIN_IDS`: ваш Telegram ID
6. Нажмите "Deploy"

## 📊 Переменные окружения

| Переменная | Описание | Пример |
|------------|----------|--------|
| `BOT_TOKEN` | Токен бота (обязательно) | `123456:ABC-DEF1234` |
| `ADMIN_IDS` | ID администраторов через запятую | `123456789,987654321` |
| `DATABASE_PATH` | Путь к БД | `casino_bot.db` |

## 🎯 Команды

- `/balance` - Проверка баланса
